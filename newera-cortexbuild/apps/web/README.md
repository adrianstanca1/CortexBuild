# Web App

## Planned Stack
- React 19 + Vite + TypeScript.
- State: Zustand + TanStack Query.
- Styling: Tailwind + Radix primitives + shared UI kit.
- Routing: React Router data APIs + module federation readiness.

## Feature Shell
- **Workspace Shell**: nav, notifications, AI copilot entry point.
- **Operations Modules**: Projects, RFIs, Submittals, Tasks, Scheduling.
- **Financial Modules**: Budgets, Invoices, Pay Apps.
- **Field Modules**: Daily logs, punch, delivery tracking, offline sync.
- **Automation Studio**: Workflow builder UI, agent marketplace.
