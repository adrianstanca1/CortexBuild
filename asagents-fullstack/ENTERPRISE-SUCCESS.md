## 🚀 How to Run

### Quick Start (All Services)
```bash
cd asagents-fullstack

# Install all dependencies
npm run setup

# Start all services at once
npm run dev:full
```

### Individual Services
```bash
# Frontend (Next.js) - Port 3000
npm run dev

# Node.js Backend - Port 3001
npm run backend:dev

# Java Backend - Port 8080
npm run java:dev
```
