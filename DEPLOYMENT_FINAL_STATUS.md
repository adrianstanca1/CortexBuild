## ⚠️ DACĂ VEZI ERORI CU SERVICE WORKER

Dacă vezi erori precum:
- `sw.js` errors
- `@vite/client` errors
- `ERR_FAILED` pentru resurse
- Icoane lipsă

**Soluție rapidă în browser:**
```
1. Ctrl+Shift+Delete (Clear browsing data)
2. Selectează "Cached images and files"
3. Clear data
4. Refresh cu Ctrl+Shift+R
```

SAU în Chrome DevTools:
```
1. F12 → Application tab
2. Service Workers → Unregister all
3. Storage → Clear site data
4. Refresh page
```
